📘 Help Guide – Blog with Admin Panel (MERN)

🧠 Project Summary:
A full-stack blog system using React, Express, MongoDB with Admin Panel and Markdown support.

📂 Structure:
- client/ → React frontend
- server/ → Express backend

🛠 Backend Features:
- Admin Auth (JWT + bcrypt)
- MongoDB CRUD APIs for Blogs

🌐 Frontend Features:
- Public blog list + view with Markdown
- Admin dashboard to Create/Edit/Delete
- Login with session (JWT)
- Dark mode toggle (top-right)

🚀 How to Run:
1. In /server → `npm install && npm start`
2. In /client → `npm install && npm start`
3. Open frontend at `http://localhost:3000`

🛡 Notes:
- Use `/auth/register` to create your first admin user
- Requires MongoDB running locally
